

# Proyecto modular

En este proyecto vamos a trabaja en un proyecto combnando los arios conocimientos que hemos ido adquiriendo a lo largo del ciclo.

La aplicacion deberá cumplir ciertas condiciones:

    * consultar actividades;
    * ver plazas disponibles;
    * reservar plaza;
    * consultar sus reservas;
    * cancelar reservas;
    * registrar incidencias.


Este proyecto esta siendo realizado por Nauzet Torres y Alberto Carballo.
